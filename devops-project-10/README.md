# Aplicación Web en Go

Este es un sitio web simple escrito en Golang. Utiliza el paquete `net/http` para servir solicitudes HTTP.

## Ejecutar el servidor

Para ejecutar el servidor, ejecuta el siguiente comando:

```bash
go run main.go
```

El servidor comenzará en el puerto 8080. Puedes acceder a él navegando a `http://localhost:8080/courses` en tu navegador web.
